# Music folder

Place audio files here to make them available in the **Audio** dropdown inside the app.

Supported formats: `.mp3`, `.wav`, `.ogg`, `.flac`, `.m4a`, `.aac`

Audio is looped automatically if it is shorter than the video, and trimmed if longer.
A 2-second fade-out is applied at the end. ffmpeg must be on your PATH for audio to work.

Download a track, drop the file into this folder, and it will appear in the dropdown next time you launch the app.
